# Curriculum Web - Jorge Lozano Fortes

Currículum web interactivo desarrollado con Vue 3 (Composition API) y Tailwind CSS.

## Características

- 🌙 Toggle modo oscuro/claro persistente (localStorage)
- 📱 Diseño totalmente responsivo
- ✨ Transiciones suaves en carga de secciones
- 🎯 Efectos parallax en el header
- 📝 Datos separados en JSON para fácil edición

## Secciones

- Header con nombre, rol y foto
- Perfil profesional
- Experiencia laboral
- Educación
- Stack tecnológico con iconos
- Certificaciones
- Idiomas
- Contacto

## Tecnologías

- Vue 3 (Composition API)
- Tailwind CSS 4
- Vite

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
```

Los archivos compilados se generan en `dist/`.
